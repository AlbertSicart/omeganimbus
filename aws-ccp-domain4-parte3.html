01 · Planes de soporte AWS — Tabla comparativa completa

Basic · Developer · Business · Enterprise On-Ramp · Enterprise

Basic
Gratis
Documentación, whitepapers, foros. Soporte de facturación y cuenta. Dashboard de estado del servicio.
Trusted Advisor: 7 checks · Sin soporte técnico · Sin acceso API de Support
Developer
Desde $29/mes
Email en horario laboral. Casos ilimitados. Orientación general de arquitectura.
Trusted Advisor: 7 checks · Sin API de Support · Personal Health Dashboard incluido
Business
Desde $100/mes
Teléfono + chat 24/7. SLA 1h urgente. Orientación completa de casos prácticos. Soporte de software de terceros.
Trusted Advisor: todos los checks · API de Support incluida · Infrastructure Event Management (tarifa adicional)
Enterprise On-Ramp
Desde $5.500/mes
Acceso a pool de TAMs (Technical Account Managers). SLA 30 min crítico. Revisiones proactivas de arquitectura.
Trusted Advisor completo · API de Support · Programas proactivos incluidos
Enterprise
Desde $15.000/mes
TAM designado dedicado. SLA 15 min crítico. Infrastructure Event Management incluido. Revisiones de arquitectura y operaciones exclusivas.
Trusted Advisor completo · API de Support · Todos los programas proactivos
Preguntas de examen más frecuentes sobre soporte
¿Plan más rentable con API de Support? → Business (más barato que Enterprise).
¿Plan más rentable con casos ilimitados? → Developer.
¿Trusted Advisor completo? → desde Business.
¿TAM designado? → solo Enterprise (Enterprise On-Ramp tiene acceso a pool, no TAM dedicado).
¿Infrastructure Event Management incluido? → solo Enterprise. Business lo tiene por tarifa adicional.

03 · Canales de soporte y servicios profesionales

TAM · AWS IQ · APN · Managed Services · Abuse

Canal / ServicioQué ofreceCuándo usarlo
TAM (Technical Account Manager) Punto de contacto designado que coordina el acceso a programas proactivos, revisiones de arquitectura y recursos de AWS. Solo Enterprise. Cuando necesitas un experto AWS dedicado que conozca tu entorno y guíe la estrategia técnica.
AWS IQ Conecta con expertos certificados de AWS de terceros para proyectos concretos. Pago por proyecto. Cuando necesitas ayuda puntual de un experto para un proyecto específico sin plan Enterprise.
AWS Partner Network (APN) Red global de socios tecnológicos y de consultoría certificados que construyen sobre o con AWS. Cuando buscas un partner externo para diseñar, migrar o gestionar entornos AWS.
AWS Managed Services (AMS) AWS gestiona las operaciones de infraestructura por ti (parcheo, monitorización, conformidad, etc.). Para empresas que quieren externalizar las operaciones cloud con garantías de SLA y conformidad.
AWS Abuse Canal para reportar uso abusivo de infraestructura AWS (spam, ataques, malware, phishing alojado en AWS). Cuando detectas actividad maliciosa originada desde infraestructura de AWS.
AWS Activate Herramientas, créditos y recursos gratuitos diseñados específicamente para startups. Para startups en etapa temprana que quieren comenzar en AWS con soporte y créditos.

▲ Exam Tips — Puntos clave
  • API de Support disponible desde el plan Business. Es la respuesta a "plan más rentable con API de Support".
  • Casos ilimitados desde el plan Developer (más económico que Business).
  • Trusted Advisor completo (todos los checks) → desde Business. Basic y Developer solo 7 checks.
  • TAM designado exclusivo → solo plan Enterprise. On-Ramp tiene acceso a pool de TAMs.
  • Infrastructure Event Management incluido → solo Enterprise. Business lo tiene por coste adicional.
  • AWS re:Post = comunidad de Q&A. Sustituto mejorado de los foros de AWS.
  • AWS IQ = expertos certificados externos para proyectos puntuales. APN = red de partners certificados.
  • AWS Abuse = reportar actividad maliciosa originada en AWS (spam, ataques, phishing…).
  • Todos los planes incluyen: documentación, whitepapers, foros, soporte de facturación y Personal Health Dashboard.
