01 · Herramientas de administración de costes

Cost Explorer · Budgets · CUR · Trusted Advisor · Calculadora

📈AWS Cost Explorer
Visualiza y analiza tendencias de gasto histórico. Proyecta costes futuros. Filtra por servicio, región, etiqueta, cuenta…
Para informes financieros interactivos y análisis a alto nivel. Permite hacer drill-down en el detalle.
🔔AWS Budgets
Define presupuestos de coste, uso o reservas. Envía alertas (SNS, email) cuando el gasto real o proyectado supera el umbral.
Para control proactivo del gasto. Alarmas de nivel gratuito y alertas automáticas por umbrales.
📋Cost and Usage Report (CUR)
El informe más detallado disponible. Desglosa costes por hora, día, mes, producto, recurso, etiqueta. Se entrega en S3.
Para análisis profundo con Athena, Redshift o QuickSight. Integración con herramientas de BI.
💡AWS Trusted Advisor
Analiza la cuenta y hace recomendaciones automáticas en 5 categorías, incluyendo optimización de costes.
Identifica instancias sobredimensionadas, volúmenes EBS infrautilizados, IPs elásticas sin usar, etc.
🧮Calculadora de precios AWS
Estima el coste mensual de servicios AWS para un caso concreto antes de desplegarlo.
Para planificación previa a la migración o evaluación de nuevas arquitecturas.
🧾AWS Billing Conductor
Servicio de facturación personalizado para partners y clientes empresariales. Soporta análisis de costes y reintegro (chargeback).
Para MSPs o empresas que necesitan redistribuir costes entre departamentos con precios personalizados.
Cost Explorer vs CUR — Cuándo usar cada uno
Cost Explorer: interfaz visual, informes interactivos a alto nivel, drill-down por dimensiones. Respuesta rápida.
CUR: máximo detalle granular (hasta por hora y por recurso). Para integraciones con Athena, Redshift o QuickSight.
02 · Gestión centralizada

AWS Organizations · Control Tower · Facturación unificada

Beneficios de la facturación unificada (Consolidated Billing)
AWS Organizations trata todas las cuentas de la organización como una sola a efectos de facturación. Beneficios:

· Descuentos por volumen compartidos: el uso agregado de todas las cuentas alcanza los umbrales de precios escalonados antes (EC2, S3…).
· Reservas compartidas: las Instancias Reservadas compradas por cualquier cuenta benefician a toda la organización.
· Vista centralizada: una sola factura con el desglose por cuenta.
· Control Tower: gobernanza multi-cuenta con guardarraíles de seguridad y conformidad preconfigurados.
03 · Visibilidad de costes

Etiquetas de asignación de costes

Cost Allocation Tags
Las etiquetas se aplican a recursos AWS (EC2, S3, RDS…) para atribuir costes a proyectos, equipos, entornos o clientes. Pasos:

1. Crear etiquetas en los recursos (ej. Project: Alpha, Team: Backend).
2. Activar las etiquetas como "etiquetas de asignación de costes" en la consola de Billing.
3. Filtrar vistas en Cost Explorer o CUR por esas etiquetas.

Son fundamentales para que los equipos sean conscientes y responsables de los costes que generan.
04 · Optimización de almacenamiento

S3 Lifecycle · Intelligent-Tiering · EBS

FunciónQué haceCuándo usarla
S3 Lifecycle Policy Mueve automáticamente objetos a una clase de almacenamiento más barata (o los elimina) tras un número de días configurado. Cuando sabes el patrón de acceso: activo los primeros 30 días, luego archivado en Glacier.
S3 Intelligent-Tiering Mueve automáticamente cada objeto al nivel de acceso más rentable según cómo se accede a él. Sin necesidad de definir reglas. Cuando el patrón de acceso es impredecible o variable. Pequeña comisión de supervisión por objeto.
S3 Requester Pays El solicitante (quien descarga) paga los costes de transferencia y solicitud, no el propietario del bucket. Cuando compartes datos con terceros y no quieres asumir los costes de transferencia.
Amazon Data Lifecycle Manager Automatiza la creación, retención y eliminación de snapshots de EBS. Reduce costes de almacenamiento de snapshots obsoletos. Para gestionar el ciclo de vida de snapshots EBS sin intervención manual.
AWS Backup Centraliza y automatiza backups de múltiples servicios. Elimina la necesidad de scripts personalizados. Para cumplir requisitos de conformidad de backups en toda la organización desde un único lugar.
05 · Migración de datos — Decisión de herramienta

Snowball vs Direct Connect vs DataSync vs Transfer Family

HerramientaVolumenMétodoCuándo elegirla
AWS Snowball Edge TB a PB Dispositivo físico Migración grande + segura + rápida. Red disponible insuficiente. Con capacidad de compute en el edge.
AWS Snowmobile PB a EB Camión físico Migraciones masivas de datacenter completo. Solo cuando el volumen es extremo.
AWS Direct Connect Variable Fibra privada dedicada Transferencia continua on-premises ↔ AWS. Baja latencia consistente. El coste de transferencia se incluye en el servicio.
AWS DataSync Cientos de TB Online (internet o Direct Connect) Transferencia automatizada de archivos. Hasta 10× más rápido que herramientas open source. Validación de datos incluida.
AWS Transfer Family Variable SFTP, FTPS, FTP Migración usando protocolos de transferencia de archivos estándar (SFTP/FTP) hacia/desde S3 o EFS.
Regla práctica para preguntas de migración
250 TB desde servidores internos a S3: la opción más rápida y rentable → varios Snowball Edge. Snowmobile es para PB-EB. Direct Connect tarda más en aprovisionarse. Subir por internet con 100 Mbps tomaría semanas.
▲ Exam Tips — Puntos clave
  • Cost Explorer = informes interactivos visuales. CUR = máximo detalle, para integraciones con Athena/Redshift.
  • AWS Budgets = alertas proactivas cuando el gasto supera un umbral definido.
  • Facturación unificada (Organizations) = descuentos por volumen compartidos entre todas las cuentas + una sola factura.
  • Etiquetas de asignación de costes = fundamentales para atribuir costes a proyectos o equipos.
  • S3 Lifecycle Policy = mover objetos automáticamente cuando sabes el patrón de acceso.
    S3 Intelligent-Tiering = mover automáticamente cuando el patrón es impredecible.
  • S3 Requester Pays = el que descarga paga la transferencia, no el propietario del bucket.
  • Data Lifecycle Manager = automatiza snapshots de EBS. AWS Backup = centraliza todos los backups.
  • Snowball Edge para TB-PB. Snowmobile para PB-EB. DataSync para transferencia online automatizada.
  • Calculadora de precios AWS = estimar costes antes de desplegar. No es una herramienta de facturación histórica.
